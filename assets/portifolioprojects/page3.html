Virtual Reality App

As part of an Erasmus+ funded research team spanning five European countries, We contributed to a project aimed at mitigating social phobia in public speaking. The outcome was a system comprising a VR mobile application and a web platform. The project included: a) Virtual Reality app for public speaking simulations. b) A web platform to manage scenarios and speeches.

Key Features

VR Mobile Application Easily create accounts for each church member, with personal details securely stored in Firebase.
Web Platform Allows users to create and practice speeches in customizable scenarios and supports shared scenarios for administrators and private options for users.
